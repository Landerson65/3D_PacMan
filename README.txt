*******************************************************************
TO RUN CODE:
*******************************************************************
1. Place all of the files in a Visual Studio Project
2. Install OpenGL
	a. Go to the Tools Menu
	b. Hover over NuGet Package Manager
	c. Select Manage NuGet Package Manager for Solution
	d. Search for Nupengl
	e. Double click on Nupengl.core
3. Turn on OpenMP Flag
	a. Right click on the Project name in Solution Explorer
	b. Select Properties
	c. Go to C/C++ > Language
	d. Turn on OpenMP Support
3. No command line arguments needed
4. Compile and run code
